Widgets for the Corsair Xeneon Edge.

Tiny single-page widgets built for the Xeneon's iCUE iFrame panel. Each folder here is a self-contained HTML file — no build step, no framework, just drop the URL into iCUE.

Point iCUE at a widget

  1. Serve this folder locally: python -m http.server 5556 from the repo root.
  2. Open iCUE → Xeneon Edge → Widgets → iFrame.
  3. Paste the widget URL, e.g. http://localhost:5556/spotify/.
  4. Or deploy to GitHub Pages / Cloudflare Pages and use the public URL from any device.
