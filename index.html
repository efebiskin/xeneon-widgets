Point iCUE at a widget

  1. Serve this folder: python -m http.server 5556 from the repo root.
  2. In iCUE → Xeneon Edge → Widgets → iFrame.
  3. Paste the widget URL, e.g. http://localhost:5556/clock/.
  4. Or push the repo to GitHub Pages / Cloudflare Pages for a public URL.
